Atlas Layout Guide

Atlas exposes its controls as a VoiceOver-friendly hierarchy. The first level is groups such as Presets, Global, Signal routing, Oscillators, Filters, Busses, Envelopes, LFOs, Effects, Modulations, and Macros. Inside each group, sections contain the actual buttons, choosers, sliders, and editors.

Navigation

  • Use VoiceOver interaction to enter a group or section.
  • The Presets group contains browser controls, preset loading, preview, search, bank, and category controls.
  • Oscillator sections include Load, Previous, Next, and Browse controls for wavetables. The Sample section has the matching sample controls.
  • The Signal routing group exposes oscillator routing, filter routing, bus sends, and filter serial modes.
  • The Busses group contains Bus 1, Bus 2, and Bus 3. Each bus has Chain and routing plus the same effects as the main chain.
  • The Effects group contains Chain and routing, then each effect by name.

Keyboard Shortcuts

Shortcut Action
Up Arrow / Down Arrow Small slider changes.
Page Up / Page Down Larger slider changes.
Enter Open an inline value field for the focused slider.
Backspace or Delete Reset the focused parameter to its default value.
VO Shift M Open modulation assignment for the focused parameter.
Return in inline fields Apply the typed value or modulation amount.
Escape in inline fields Cancel the typed value or modulation amount.

Preset Workflow

Open the Presets group, choose a bank or category if desired, type in Search, then select a preset. Press Enter or use Load selected preset to load it. Loading and changing presets preserves VoiceOver focus whenever possible.

Wavetables and Samples

Each oscillator has Load wavetable, Previous wavetable, Next wavetable, and Browse wavetable folders. Browse wavetable folders opens the folder tree under the Atlas wavetable paths. The Sample oscillator offers the same workflow for audio samples.

Bus Workflow

Set an oscillator, the sample oscillator, filter 1, or filter 2 destination to Bus 1, Bus 2, or Bus 3 from Signal routing or from that source section. Then open the Busses group and choose the bus to edit its effect chain. Busses can be routed to the main chain, direct out, or another bus when that does not create feedback.

Modulation Workflow

Focus a parameter and press VO Shift M. Choose a modulation source. Atlas then focuses an inline text field where you can type the starting maximum modulation amount. You can fine-tune the amount later in the Modulations group.

LFO Editor

The LFO group includes shape selection and a keyboard editor. Use comma and period to move between points, A to add a point, E to delete a point, arrows to move or edit points, minus and equals to change grid size, and semicolon or apostrophe to change cycle length. The focused editor summary shows the selected point, time, value, and curve.

Installed Atlas Folder

The installer creates ~/Documents/Alessio Plugins/Atlas for the logged-in user if it does not exist, including Factory and User folders for Presets, Wavetables, Samples, LFOs, and FX presets. This guide is copied there as Accessible Layout Guide.html.
